The pipeline

How the bot works

Five stages, each a separate, testable piece. The same strategy code runs in the backtester and live — only the data and the order destination change.

1 · Data

Prices in

Free historical bars from Yahoo for backtests; real-time IEX bars from Alpaca for live. One normalized OHLCV shape feeds everything.

2 · Strategy

A rule

An indicator turns prices into a signal — a moving average, a z-score, an RSI, a spread. Each strategy is ~40 lines and fully transparent.

3 · Backtest

Test on history

An event-driven engine that decides on today's close and fills at tomorrow's open — so it can't cheat by peeking at the future.

4 · Paper → Live

Trade for real

The same logic talks to a broker behind risk guards: a kill switch, position caps, and an explicit opt-in before real money.
